All about Background Checks

When a client calls in and asks: "How do I run background checks?"

If they've never done it before Follow these steps:

Adding Background Check Capability to a New Account:

  1. While on the phone with the customer: Get their Contact Information.
  2. Next you'll pass that info to the Head of Billing (Currently Pearl).
  3. During this time Billing has to speak with our Background Check Company called Sapphire.
  4. Billing will then send the client an agreement that they have to sign Electronically.
  5. After signing the document, Head of Billing will contact Head of Support (currently Jeffrey) to activate Background Checks.
  6. This is done by going to Setup > System > Background Check > Then Adding the Sapphie Background Check.
  7. Now Background Checks are ready to be used.
  8. Contact the client and let them know that we've added the Background Check service onto the account.
  9. Now it's done! They've gained the ability to run Background Checks on their account.
  10. Next up is using the Payroll.

Using Background Checks

So next we'll walk them through using Background Checks

  1. Take them to the Hiring Tab.
  2. Now they have to make a Job Opening. Make them select "New Opening"
  3. They fill in the information and list their job.
  4. Some of these fields can be set so that they must be filled out
  5. As they're filling out the listing, instruct them they must require a few fields:
    1. Social Security
    2. Background Check Permissions.
    3. Birthdate
    4. Name
    5. Address

    Note:

    • If the Employer Doesn't Make Permission for Background Checks Required, and the hire denies the permission, they wont be able to run the Background Check.
    • You can't run a Background Check without a Social Security Number, so it should be required, as well
    • Name and Address are also recommended fields to require for the application.
    • Be sure to tell them to make these fields required in their listing!

  6. After creating the listing, instruct them to give the Job Opening link to a prospective employee or place it where they wish, such as Craigslist or their own website.
  7. Now, when the New Hire clicks on the listing and fills it out they can run the Background Check

Running the Background Check

Once application is in, they can now run a Background Check on the Employee.

  1. They should go to the Applicants Area
  2. Click on the 3 bar Icon next to the applicant they want to run Background Checks on.
  3. Click on Run Background Checks
  4. Click on service provider. There are two packages: 17.50$ for Ruby and 40$ for Emerald
  5. The Emerald just does more checks. Checking by county as well
  6. They'll then get an automatic email from us that Sapphire is working on it
  7. The bg check is usually completed really quickly

If the Employee is an existing Employee they wish to run a Background Check for:

  1. Go to Hiring Tab
  2. Go to Job Openings
  3. Click 3 bars Icon for the job.
  4. Click View applicants.
  5. Click add application
  6. Fill in all the fields as if they were the Employee
  7. Click give Background Check Permissions
  8. Run the Background Check as Normally
